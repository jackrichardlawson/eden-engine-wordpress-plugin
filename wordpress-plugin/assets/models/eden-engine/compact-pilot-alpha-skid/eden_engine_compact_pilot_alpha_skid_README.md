# Eden Engine Compact Pilot / Alpha Skid 3D Model

This package contains a conceptual 3D replica of a compact Eden Engine pilot/alpha skid, sized similarly to the current 3D twin screenshot rather than a full shipping-container unit.

## Files

- `eden_engine_compact_pilot_alpha_skid.glb` — primary 3D model with object names and color-coded modules.
- `eden_engine_compact_pilot_alpha_skid.obj` — alternate OBJ export for tools that prefer OBJ. Color support varies by viewer.
- `eden_engine_compact_pilot_alpha_skid_manifest.json` — module map with IDs, zones, maturity levels, dimensions, and notes.

## Model Intent

The model shows a compact pilot-scale process skid with:

1. CO2 source and conditioning
2. CO2 electrolyzer stack
3. Electrolyte loop
4. Carbonate / salt management
5. Formate buffer tank
6. Formate shuttle / salt firewall
7. Formate Activator breakthrough module
8. C1 capture cartridge
9. Sugar builder cartridge rack
10. Product recovery membrane skid
11. Syrup collection / storage tank
12. Power electronics, thermal loop, controls, sensors, and safety interlock support systems

## Safety / Reality Notes

- Concept visualization only.
- Not a fabrication drawing.
- Not validated hardware.
- Real hardware execution is not enabled.
- Dry-run planning only.
- The Formate Activator is intentionally shown as a red breakthrough placeholder, not solved chemistry.

## Viewer Tips

Import the `.glb` into Blender, Three.js, Sketchfab, or most modern CAD/3D viewers. Object names are descriptive so you can select modules individually in the scene tree.
